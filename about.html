About Book-Explorer

Web App

A responsive,intersctive,and IPA-powered book browsing App built across four labs
Covering HTML/CSS with Tailwind, JavaScript modules,async API integration,and
GitHub collaboration workflows.

Search millions of books
Responsive on all devices
Add favorites to localstorage
Open Library API
Multi-page Navigation

lab Breakdown

01

Responsive Multi-Page Layouts

Home page with navbar,hero section,and footer using Tailwind .Responsive grid adapts from 2 colums on mobile to 6 on large screens.

02

DOM Interactivity & JavaScript Modules

favorites page with add/remove fuction ality backed by modular helper functions and DOM event handers wired to vanilla JS state

03

Async JavaScript & API Integration

fetches real book from the open Library API using async/await.Includes search bar.

04

GitHub Collaboration

Built for collaborative workflow :feature branches for search and favorites functionality.

Tech Stack

  • Frontend: HTML, CSS, JavaScript
  • Date: Open Library Rest API
  • icons: Font Awesome 6
  • Styling: Tailwind CSS
  • persistence: LocalStorage
  • font: Google Fonts
