# Book Explorer

A small multi-page book search app built with HTML, Tailwind CSS, and JavaScript.

## Pages

- `Html/index.html` — Home page with search, book cards, and favorite actions.
- `Html/favorites.html` — Favorites page showing saved books.
- `Html/about.html` — About page describing the project.

## Features

- Search books using the Open Library API
- Save favorites to `localStorage`
- Responsive mobile navigation
- Toast notifications for add/remove actions
- Favorites counter displayed in the navigation

## How to run

1. Open `Html/index.html` in your browser.
2. Use the search box to find books.
3. Click the favorite button on a book card to save it.
4. Visit `Html/favorites.html` to see your saved books.

## Notes

- The app expects the JavaScript files under `Asset/`.
- Favorites are stored locally in the browser using `localStorage`.
